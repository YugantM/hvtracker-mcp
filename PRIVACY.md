# Privacy

HVTracker MCP sends the search string, package name, repository, display name,
or MCP server URL you provide to `https://hvtracker.net` so it can return public
trust data.

The server does not require an API key, does not write to your system, and does
not intentionally collect files, credentials, or chat transcripts. Network logs
for requests to `hvtracker.net` may be processed by the hosting provider and
application infrastructure for security, reliability, and abuse prevention.
